# Harshdeep Parmar - Portfolio

Personal portfolio website showcasing my work in embedded systems, edge computing, music technology, and photography.

🌐 **Live Site:** [hdparmar.github.io](https://hdparmar.github.io)

## About

I'm a research-oriented entrepreneur specializing in embedded systems, edge computing, and secure partitioning. Currently building [Tonestruments](https://tonestruments.com) — helping people learn music production through playful, gamified devices.

## Sections

- **Tonestruments** — My music tech startup building TonePad
- **Research** — Academic work on AI-generated music (Tradifusion)
- **Writing** — Essays and reflections
- **Photography** — Film photography collection

## Tech Stack

- **Framework:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Deployment:** GitHub Pages via GitHub Actions

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

Pushing to `main` triggers automatic deployment to GitHub Pages via GitHub Actions.
